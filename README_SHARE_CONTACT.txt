SHARE CONTACT UPDATE

- Button changed from "Save to Contacts" to "Share Contact".
- "Share Contact" shares the .vcf contact file via the native share sheet (supported iOS/Android).
- Fallback: opens/downloads the .vcf file.
- Tracks share taps via /go/share/<slug>.

NFC NOTE
- Program NFC tags to the tracked URL:
  https://<your-domain>/go/nfc/<slug>
